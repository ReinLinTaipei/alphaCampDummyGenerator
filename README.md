# Dummy Generator

This project is a dummy generator as title.
You can choose one of offered careers for fun.

![website snapshot](./public/img/dummy_generator_screenshot.png)

**Prerequisites**

> - Node.js `v10.16.0`

**Installation**

> - git clone project
>   ```
>   git clone http://github.com/reinlintaipei
>   ```
> - install package
>   ```
>   npm install
>   ```
> - run the app
>   ```
>   node app.js
>   ```
> - open browser
>   ```
>    http://localhost:3000
>   ```
> - enjoy ~
